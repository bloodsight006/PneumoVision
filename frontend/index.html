    About PneumoVision

    An AI-powered chest X-ray analysis tool built for academic research. Detects COVID-19 and Pneumonia using deep learning with explainability via Grad-CAM.

    Model

    DenseNet121 fine-tuned on COVID-19 Radiography Database (21K images). Two-phase transfer learning with class-weighted loss achieves ~96–98% validation accuracy.

    Grad-CAM

    Gradient-weighted Class Activation Mapping produces a visual heatmap of which lung regions influenced the model's decision — enabling radiologist-level interpretability.

    BLAA Novelty

    Bilateral Lung Asymmetry Analysis splits the GradCAM heatmap into left/right halves and scores each independently. COVID-19 typically shows bilateral symmetric infiltrates; bacterial pneumonia shows unilateral/lobar consolidation — giving a second differential signal.

    Stack

    Frontend: HTML5 · CSS3 · Vanilla JS
    Backend: Python · FastAPI
    ML: TensorFlow · DenseNet121
    DB: Firebase Firestore
